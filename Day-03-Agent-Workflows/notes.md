# Day 3: Agent Skills

## Overview

Today I learned how AI agents can be extended using reusable skills. Instead of storing all knowledge and instructions inside a single prompt, agents can load specialized skills when needed.

## Key Concepts Learned

### Agent Skills

Agent Skills are reusable capabilities that help agents perform specialized tasks. Skills can contain instructions, tools, workflows, and domain-specific knowledge.

### Context Rot

As prompts become larger, AI systems may become less efficient and less focused. Context rot occurs when excessive information reduces performance and increases complexity.

### Progressive Disclosure

Progressive disclosure allows agents to load only the information required for a specific task. This keeps prompts smaller and improves efficiency.

### Specialized Agents

By combining different skills, a single agent can perform multiple roles such as coding, research assistance, document analysis, and data processing.
