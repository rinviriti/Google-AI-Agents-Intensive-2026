# Day 2: Agent Tools & Interoperability

## Overview

Today I learned how AI agents connect to external tools, services, and data sources through standardized protocols. The main focus was understanding the Model Context Protocol (MCP) and agent interoperability.

## Key Concepts Learned

### Model Context Protocol (MCP)

MCP is a standardized protocol that allows AI agents to communicate with external tools and services. It helps eliminate the need for custom integrations between every model and every tool.

### Agent Interoperability

Interoperability enables agents to work with different systems, services, and resources using common standards and protocols.

### Agent-to-Agent (A2A)

A2A communication allows multiple AI agents to collaborate and exchange information while working toward a common objective.

### Agent-to-User Interface (A2UI)

A2UI enables agents to dynamically generate interfaces and interactive experiences for users.
