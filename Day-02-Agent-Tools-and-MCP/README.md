# Day 2: Agent Tools & Interoperability

## Topics Covered
- Model Context Protocol (MCP)
- Tool use in AI agents
- Agent interoperability
- Agent-to-Agent communication
- Agent-to-User Interface
- MCP servers in Antigravity

## What I Learned
Today I learned that AI agents become more useful when they can connect to external tools, data sources, APIs, and documentation through standard protocols such as MCP.

MCP acts like a bridge between AI models and external tools, allowing agents to access information and perform actions in a structured way.
